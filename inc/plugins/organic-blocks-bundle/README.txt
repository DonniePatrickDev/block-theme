A collection of powerful blocks for the Gutenberg editor.

== Description ==

Organic Blocks Bundle includes a powerful collection of Gutenberg blocks.

* Header Block
* Footer Block
* Content Slideshow Block
* Testimonials Block
* Portfolio Block
* Posts Block
* Profile Block
* Widget Area Block
* Toggle Block
* Alert Block
* Modal Block
* Callout Block
* Featured Content Block
* Hero Block
* Icon Box Block
* Pricing Table Block
* Position Container Block
* Max-Width Container Block
* Link Container Block
* Copy To Clipboard Block

= Getting Started =

Follow the simple steps below to begin using Organic Blocks:

1. Navigate to any page or post using the Gutenberg block editor.
2. Within the editor, click the "+" button to add a new block.
3. Search "Organic" within the list of available blocks.
4. Choose your desired block.
5. Setup the block as instructed.

== Installation ==

Note: This plugin relies on new WordPress features, so it requires WordPress version 5.2+ and PHP 7.0+

1. Upload 'organic-blocks-bundle' to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to any post or page using the Gutenberg block editor.
4. Search "Organic" to see all available Organic Blocks.
